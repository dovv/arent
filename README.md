# README

## Requierdment
```
- Rails: 7.0.5
- Ruby: 2.7.2
- Postgres: >= 9
```

## Init
```
- bundle install
- yarn install && yarn run build
- rails db:create
- rails db:migrate
- rails db:seed
```

## Run local
```
foreman start -f Procfile.dev "$@"
http://127.0.0.1:3000
```
